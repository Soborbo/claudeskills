# Long-tail Keyword Modifiers

## Location Modifiers

| Type | Examples |
|------|----------|
| City | `bristol`, `bath`, `london` |
| Area | `south west`, `city centre`, `near me` |
| Postcode | `BS1`, `BA1` |

## Service Modifiers

| Type | Examples |
|------|----------|
| Urgency | `emergency`, `same day`, `urgent`, `24 hour` |
| Quality | `best`, `professional`, `trusted`, `reliable` |
| Price | `cheap`, `affordable`, `budget`, `cost` |
| Comparison | `vs`, `or`, `alternative` |

## Intent Modifiers

| Intent | Modifiers |
|--------|-----------|
| Commercial | `services`, `company`, `near me`, `hire` |
| Informational | `how to`, `what is`, `guide`, `tips` |
| Transactional | `quote`, `book`, `price`, `cost` |

## Industry-Specific (UK Services)

### Removals
- `man and van`
- `house removals`
- `office removals`
- `packing service`

### Trade Services
- `emergency callout`
- `free quote`
- `no call out fee`
- `fixed price`

## Combination Patterns

```
[service] + [location]
[modifier] + [service] + [location]
[service] + near me
[service] + [location] + [modifier]
```

## Examples

| Base Keyword | With Modifiers |
|--------------|----------------|
| removals | `cheap removals bristol` |
| plumber | `emergency plumber near me` |
| electrician | `24 hour electrician bath` |
