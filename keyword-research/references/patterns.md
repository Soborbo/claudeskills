# Keyword Patterns

## Canonical Keyword Patterns

### Homepage

```
{core_service} {location}
```

| Example | Notes |
|---------|-------|
| "house removals bristol" | Service + city |
| "removal company bristol" | Variation |
| "movers bristol" | Short form |

### Service Page (Silo Top)

```
{specific_service} {location}
```

| Example | Notes |
|---------|-------|
| "office removals bristol" | Specific service |
| "packing service bristol" | Specific service |
| "furniture storage bristol" | Specific service |

### Area Page

```
{service} {area}
{service} in {area}
removals {area}
```

| Example | Notes |
|---------|-------|
| "removals bath" | Short, common |
| "house removals in bath" | Long form |
| "movers weston-super-mare" | With hyphen |

### Article Page

```
how much {does/do} {service} cost
how to {action} {object}
{option A} vs {option B}
{number} {tips/ways} to {outcome}
```

| Example | Notes |
|---------|-------|
| "how much do removals cost" | Cost guide |
| "how to pack fragile items" | How-to |
| "removal company vs man and van" | Comparison |
| "10 tips for stress-free moving" | List |

## Supporting Keyword Patterns

| Example |

| "local moving company bristol" |

```
{synonym} {area}
{service} near {area}
{area} {service_noun}
```

| Example |

## Question Patterns

### Cost Questions

```
how much do {service} cost
how much does it cost to {action}
what is the average cost of {service}
{service} prices {location}
```

### Process Questions

```
how long does {service} take
what do {service} companies do
how does {service} work
what to expect from {service}
```

### Comparison Questions

```
{option A} vs {option B}
{option A} or {option B}
should I {option A} or {option B}
difference between {A} and {B}
```

### Booking Questions

```
when to book {service}
how far in advance to book {service}
how to choose a {service} company
what to look for in {service}
```

## Generation Rules

### For New Business

1. Start with core service + primary location
2. Generate area variants for each service_area
3. Add 2-3 synonyms per service
4. Apply modifiers (1-2 types per page)
5. Generate questions (6-10 for FAQ)

### Keyword Count Targets

| Site Size | Silos | Keywords |
|-----------|-------|----------|
| Small (5 pages) | 1 | 15-20 |
| Medium (10 pages) | 2 | 30-40 |
| Large (20+ pages) | 3+ | 50+ |
