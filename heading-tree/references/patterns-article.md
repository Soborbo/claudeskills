# Article Page Heading Patterns

## H1

**Pattern:** `{Keyword-Rich Question/Title}`

| Example | Type |
|---------|------|
| "How Much Does a House Move Cost in 2024?" | Question |
| "Complete Guide to Moving House in Bristol" | Guide |
| "10 Tips for a Stress-Free House Move" | List |
| "House Removals vs Man and Van: Which to Choose?" | Comparison |

**Rules:**
- Keyword-rich but natural
- Include year if time-sensitive
- Max 60 characters for SEO

## H2 Sections

Main content divisions. 4-7 typical.

```
H1: How Much Does a House Move Cost in 2024?
  H2: Factors That Affect Moving Costs
  H2: Average UK Removal Prices
  H2: How to Get the Best Price
  H2: Hidden Costs to Watch For
  H2: DIY vs Professional: Cost Comparison
  H2: Frequently Asked Questions
```

## H3 Subsections

Detailed breakdowns within H2s. 2-5 per H2.

```
H2: Factors That Affect Moving Costs
  H3: Property Size
  H3: Distance
  H3: Time of Year
  H3: Access Difficulties
  H3: Special Items
```

## Article Type Variations

### How-To Guide

```
H1: How to [Do Something]
  H2: What You'll Need
  H2: Step 1: [First Step]
  H2: Step 2: [Second Step]
  H2: Common Mistakes to Avoid
  H2: FAQs
```

### Comparison Article

```
H1: [Option A] vs [Option B]: Which is Better?
  H2: Overview
  H2: [Option A] Pros and Cons
  H2: [Option B] Pros and Cons
  H2: Cost Comparison
  H2: Which Should You Choose?
  H2: FAQs
```

### List Article

```
H1: [Number] [Things] for [Outcome]
  H2: 1. [First Item]
  H2: 2. [Second Item]
  H2: 3. [Third Item]
  ...
  H2: Conclusion
```

## H4 Rules Summary

| Rule | Requirement |
|------|-------------|
| Allowed pages | Article ONLY |
| Max per article | 5 |
| Must have parent | H3 (never orphaned) |
| Forbidden | Landing, service, service-area |
