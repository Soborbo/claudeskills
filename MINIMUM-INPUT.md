# Minimum Input Requirements

Do NOT start work without these inputs. If missing: STOP → List missing items → ASK.

## Required Before ANY Work

| Item | Example | Why Required |
|------|---------|--------------|
| Business name | "Smith & Sons Removals" | Used everywhere |
| Primary service/product | "House removals" | Core messaging |
| Target location | "Manchester, UK" | Local SEO, content |
| Phone number | "+44 161 123 4567" | Contact, schema |
| Email address | "info@example.com" | Forms, contact |
| Primary conversion goal | "Quote request" | CTA focus |

**If any missing:** Cannot proceed. These define the entire project.

## Required Before Design

| Item | Example | Why Required |
|------|---------|--------------|
| Brand colors (hex) | Primary: #1E40AF, Secondary: #F59E0B | Tailwind config |
| Logo file | logo.svg (vector preferred) | Header, footer, favicon |
| Competitor URLs | 2-3 competitor websites | Design direction, differentiation |

**If missing:** Can proceed with placeholder, but flag for replacement.

## Required Before Forms Work

| Item | Example | Why Required |
|------|---------|--------------|
| Privacy policy URL | /privacy | Form compliance |
| GDPR consent text | "I agree to the privacy policy..." | Legal requirement |
| Form recipient email | leads@example.com | Where submissions go |
| Thank-you message | "Thanks! We'll call within 24h" | Post-submit UX |
| Spam protection | Turnstile site key | Form security |

**If missing:** Form cannot go live. Build with placeholders, block deployment.

## Required Before Content

| Item | Example | Why Required |
|------|---------|--------------|
| Service descriptions | 3-5 sentences per service | Service pages |
| Unique selling points | "Family-run since 1985" | Messaging |
| Testimonials | 3-5 real reviews with names | Social proof |
| FAQ content | 5-10 common questions | FAQ section, schema |

**If missing:** Can use placeholder, must replace before launch.

## Required Before Launch

| Item | Example | Why Required |
|------|---------|--------------|
| Domain name | example.com | Deployment |
| GTM/GA4 ID | GTM-XXXXXX | Analytics |
| Privacy policy content | Full legal text | Legal compliance |
| Cookie consent config | Categories, descriptions | GDPR compliance |
| Turnstile keys | Site key + secret | Form protection |
| Email API key | Resend/Brevo key | Form delivery |

**If missing:** Cannot deploy. Site not production-ready.

## Optional But Recommended

| Item | Fallback | Impact |
|------|----------|--------|
| Team photos | Stock/illustration | Less personal |
| Project gallery | Stock images | Less credible |
| Video content | None | Lower engagement |
| Blog posts | None | Less SEO value |
| Social media links | Omit section | Less trust signals |

## What To Do When Input Missing

```
1. STOP current task
2. List ALL missing items (not one at a time)
3. Explain WHY each is needed
4. Provide examples of what's needed
5. Wait for response
6. Do NOT use Lorem Ipsum or placeholder.com
7. Do NOT invent business details
8. Do NOT proceed with assumptions
```
