# Maintenance Agreement Template

**Note:** This is a reference template only. Customize terms and pricing per project.

```markdown
# Website Maintenance Agreement

## Included in Monthly Maintenance (£XX/month)

### Technical Maintenance
- Security updates and patches
- Performance monitoring
- Uptime monitoring (99.9% target)
- SSL certificate management
- Daily backups

### Minor Updates (up to X hours/month)
- Text corrections
- Contact info updates
- Image swaps (provided by client)
- Link updates

### Support
- Email support (response within 24h)
- Monthly performance report

## Not Included (Quoted Separately)

- New pages or sections
- Design changes
- New features or functionality
- Third-party integrations
- Content writing
- Photography/graphics
- SEO campaigns
- Advertising management

## Emergency Support

- Critical issues: Response within 4 hours
- Definition: Site completely down or forms not working
- Available: Monday-Friday, 9am-6pm UK

## Payment Terms

- Monthly invoicing
- Due within 14 days
- 30 days notice to cancel

## Term

- Minimum 3 months
- Then rolling monthly
```

## Customization Points

| Field | Typical Values |
|-------|----------------|
| Monthly price | £50-150/month |
| Hours included | 1-3 hours/month |
| Response time | 24h normal, 4h emergency |
| Minimum term | 3-6 months |
| Notice period | 30 days |

## What Counts as "Emergency"

| Emergency ✅ | Not Emergency ❌ |
|-------------|-----------------|
| Site completely down | Typo on page |
| Forms not sending | Want new feature |
| SSL certificate error | Design change request |
| Security breach | Content update |
